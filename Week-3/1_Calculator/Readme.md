# Write a smart contract in Solidity that implements a basic calculator with the following features:

1. Addition: a function that takes two integers and returns their sum.
2. Subtraction: a function that takes two integers and returns their difference.
3. Multiplication: a function that takes two integers and returns their product.
4. Division: a function that takes two integers and returns their quotient.
5. Modulo: a function that takes two integers and returns the remainder of their division.
